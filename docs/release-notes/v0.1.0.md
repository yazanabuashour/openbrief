# OpenBrief v0.1.0

This first public release ships the local-first OpenBrief JSON runner and single-file agent skill for configurable RSS, Atom, and GitHub release brief workflows.

## Changed

- Added the production `openbrief config` and `openbrief brief` runner contract with SQLite-backed sources, outlet policies, feed state, health warnings, and delivery history.
- Added the shipped `skills/openbrief/SKILL.md` agent policy for runner-only brief and configuration workflows.
- Added generic feed processing for URL canonicalization, outlet extraction, outlet policy audit or suppression, source priority, always-report sources, same-run topic deduplication, and recent-delivery suppression.
- Added public release assets, release verification documentation, security and maintainer runbooks, OSS metadata, and production agent eval evidence for the v0.1.0 gate.

## Verification

- Local release gates passed: release-doc validation, skill validation, formatting check, golangci-lint, unit tests, and committed-doc policy scrub.
- The production OpenBrief agent eval gate passed the selected v0.1.0 scenarios with neutral `<run-root>` artifact references.
