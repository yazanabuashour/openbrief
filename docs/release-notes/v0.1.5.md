# OpenBrief v0.1.5

This patch release improves Google News and feed canonicalization reliability without changing the runner CLI, JSON schema, SQLite schema, or install flow.

## Changed

- Preserve feed candidates when URL canonicalization fails so transient resolver errors do not hide otherwise usable feed items.
- Memoize duplicate Google News article URL resolution within a single brief run to avoid resolving the same article URL repeatedly.
- Back off Google News article resolution after HTTP 429 or timeout signals so follow-up resolver requests slow down while healthy sources can still produce candidates.
